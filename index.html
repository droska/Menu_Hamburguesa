Una de las tendencias web que más se ha usado este año es la de utilizar menús hamburguesa. Este nombre es el que se le ha dado a este tipo de navegación, cuya funcionalidad permite que los elementos de la misma se mantengan ocultos, dejando así más espacio para el contenido de nuestra web, y mostrándolos solo al realizar alguna interacción con él.
